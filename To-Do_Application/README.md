# To-Do List Application

A simple and responsive **To-Do List App** built with **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, event handling, and persistence using **localStorage**.

---

## Features

- Add new tasks with a single click.
- Mark tasks as **completed** by clicking on them.
- Delete tasks individually using the **Delete** button.
- Tasks are saved in **localStorage** and persist after page refresh.
- Responsive design with a clean UI.

---

## Project Structure

- To-Do_Application/
- To-Do.html # Main HTML file
- To-Do.css # Styling for the app
- To-Do.js # JavaScript logic

## Functions :

1. **Add Task**
2. **Mark Completed**
3. **Delete Task**
