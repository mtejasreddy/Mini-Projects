# Weather App

A simple and user friendly weather app built with **HTML, CSS and JavaScript**. This mini project uses DOM manipulation, event handling and fetching of **API's**. A simple weather app that fetches real-time weather data using the **OpenWeather API**.

---

## Features

- Fetches weather report with a single click.
- Displays **Temperature, Condition and Image**.
- Used **API's** for fetching the Report of weather.
- User friendly and with clean UI.
- Error handling for invalid city names.

---

## Project Structure

- Weather Application/
- weather.html # Main HTML file
- weather.css # Styling for the app
- weather.js # JavaScript logic
