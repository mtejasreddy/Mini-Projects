# 🎯 Quiz App

A simple and interactive Quiz Application built with **ReactJS**.  
This project is part of my Mini-Projects portfolio, designed to practice frontend development, API handling.

---

## 🚀 Features

- Multiple-choice questions with instant feedback
- Score tracking at the end of the quiz
- Responsive design for desktop and mobile
- Clean UI built with React components
- Easy to extend with new questions

---

## 🛠️ Tech Stack

- **ReactJS** (Frontend framework)
- \*\*JavaScript
- **CSS3** for styling
- **React Scripts** for build and development
