# Responsive Portfolio Page

A simple, beginner-friendly responsive portfolio website built with HTML and CSS.

## Features

-- Responsive design for desktop and mobile
<br>
-- Simple navigation menu (Home, About, Skills, Projects, Contact)
<br>
-- Social media links (LinkedIn, GitHub, LeetCode)
<br>
-- Grid layout for projects
<br>
-- Contact section with email
<br>
-- Footer with copyright

## File Structure

```
Responsive_Portfolio_Page/
├── Portfolio.html      # Main HTML file
├── Portfolio.css       # Stylesheet
└── README.md          # This file
```

## How to Use

1. Open `Portfolio.html` in your web browser
2. Navigate through sections using the menu links
3. Update the social media links with your actual profiles
