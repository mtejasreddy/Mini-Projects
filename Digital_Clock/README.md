# Digital Clock

A simple **Digital Clock** built using ** HTML, CSS, JavaScript**.
The project displays the current time in `HH:MM:SS` format and updates every second

## Features

- Real-time clock display
- styled with modern CSS.
- Clean and minimal UI

## Technologies Used

-**HTML5** -> structure of the clock

- **CSS3** -> styling and layout
- **JavaScript** -> Logic and updating time

## Project Structure

Digital_Clock/

- clock.html # Main HTML file
- clock.css # Styling for the clock
- clock.js # JavaScript logic

## Output Preview

The clock appears centered on the screen and updates every second : 12:45:09
