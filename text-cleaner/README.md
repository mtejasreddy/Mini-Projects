# Text Cleaner App

A beginner-friendly mini project that allows users to clean text by removing spaces, punctuation, and converting case.  
Built with **HTML, CSS, and JavaScript**.

---

## Features

- Remove all spaces from text
- Remove punctuation marks
- Convert text to lowercase and uppercase
- Simple, responsive UI

---

## 🛠️ Tech Stack

- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – functionality
- **Regex** – text cleaning logic

---

## 🎮 How to Use

1. Enter/paste text in the input box.
2. Select cleaning options:
   - Remove spaces
   - Remove punctuation
   - Convert to lowercase
   - Convert to Uppercase
3. Click **Clean Text**.
4. View the cleaned output instantly.
